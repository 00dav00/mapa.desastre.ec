== Description ==
Gives you the ability to do a full, 100% backup of your deployment at desired intervals.

== Installation ==
1. Copy the entire /backup/ directory into your /plugins/ directory.
2. CHMOD the /backup/warehouse/ directory to 755 or 777.
3. Activate the plugin.
